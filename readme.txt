////////////////////////////////////////////////////////////////////
S.T.A.L.K.E.R. Anomaly 1.5.3 STALKER-Anomaly-modded-exes対応日本語化MOD
////////////////////////////////////////////////////////////////////

◆概要
このMODは、S.T.A.L.K.E.R. Anomaly 1.5.3を日本語化するためのMODで、STALKER-Anomaly-modded-exesに対応しています。
ベースには、ポマギーチェどっとコム3にアップロードされていた1.5.1用日本語化MODを使用しています。
STALKER-Anomaly-modded-exesでは、マルチバイトフォント環境で文字に色を付けた際に発生していた意図しない改行バグが修正されています。
これにより、従来の日本語化MODで意図的に削除していた改行が不要になりました。

このMODを使用するには、STALKER-Anomaly-modded-exesのdpurple.diffが適用されているexeファイルが必要です。
https://github.com/themrdemonized/xray-monolith

◆注意事項
STALKER-Anomaly-modded-exesを導入していない環境でこのMODを使用すると、
ゲーム内で意図しない余計な改行が発生し、テキストが正しく表示されなくなりますのでご注意ください。

◆導入手順
1. 以下のURLからSTALKER-Anomaly-modded-exesをダウンロードしてください。
    ダウンロード方法はリンク先のページを確認してください。
    https://github.com/themrdemonized/xray-monolith

2. STALKER-Anomaly-modded-exesのページのReadmeに従って、STALKER-Anomaly-modded-exesをインストールしてください。

3. Mod Organizer 2を使用し、本MODの優先度を「0」（MODリストの一番上）に設定してください。

4. Anomalyのランチャーから「シェーダーキャッシュクリア」を行い、その後ゲームを起動してください。
